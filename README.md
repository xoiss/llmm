`llmm` is a cross-platform command-line application designed for running complex LLM scripts. It allows you to configure the LLM role in a dialog according to a predefined profile, use templates and substitutions to automatically build prompts, save and analyze dialogs, and more. It can be useful for prototyping and prompt engineering.

Requires Python 3.11 or later.
